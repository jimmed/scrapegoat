# Building from Source

## 1. Clone the repository

```bash
git clone https://github.com/jimmed/severus-scrape.git
cd severus-scrape
```

## 2. Install dependencies

```bash
npm install
```

or

```
yarn
```

## 3. Compile

```bash
npm run build
```

or

```
yarn build
```
