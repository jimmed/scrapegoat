# Building from Source

## 1. Clone the repository

```bash
git clone https://github.com/jimmed/scrapegoat.git
cd scrapegoat
```

## 2. Install dependencies

```bash
npm install
```

or

```
yarn
```

## 3. Compile

```bash
npm run build
```

or

```
yarn build
```
