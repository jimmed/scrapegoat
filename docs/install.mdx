# Installation

You may install Severus Scrape into your project using [npm](https://npmjs.com) by running the following command from your project root:

```bash
npm install severus-scrape
```

If you prefer, you may use [yarn](https://yarnpkg.com), or any other compatible package manager.

You may also [build Severus Scrape from source](./from-source).
