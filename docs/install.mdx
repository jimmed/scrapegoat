# Installation

You may install scrapegoat into your project using [npm](https://npmjs.com) by running the following command from your project root:

```bash
npm install scrapegoat
```

If you prefer, you may use [yarn](https://yarnpkg.com), or any other compatible package manager.

You may also [build scrapegoat from source](./from-source).
